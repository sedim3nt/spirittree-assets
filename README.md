# SpiritTree Assets

Shared image assets for SpiritTree content — blog covers, social media graphics, and brand materials.

**Live:** (Asset CDN via GitHub Pages)
**Stack:** Static assets
**Status:** Active

## What This Is

SpiritTree Assets is a centralized repository for images used across SpiritTree's content channels — Substack cover images, social media graphics, and brand materials. Served via GitHub Pages with a custom CNAME, it acts as a lightweight CDN for the content operation.

## Contents

- 📸 **Blog Covers** — Substack and newsletter cover images
- 🌌 **Cosmic Webb** — Webb telescope imagery for content use
- 🎨 **Brand Materials** — logos, graphics, and visual identity assets

## AI Integration

None.

## Tech Stack

- **Format:** PNG/JPEG images, HTML index
- **Hosting:** GitHub Pages (custom CNAME)
